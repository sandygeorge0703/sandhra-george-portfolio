A few of my projects...

Tin Dog

This playful Tinder-inspired app caters to Gen Z's love for dogs. Using CSS gradients and Bootstrap, I crafted a vibrant and interactive platform that lets dog enthusiasts connect in a fun and modern way. Experience a truly pawsome blend of design and functionality!

Drum Kit

This interactive Drum Kit project combines HTML, CSS, and JavaScript to deliver a fun and engaging musical experience. Users can play different drum sounds by clicking on visually appealing buttons or pressing corresponding keys on their keyboard. With smooth animations and a user-friendly interface, this project showcases the power of web technologies in creating an enjoyable rhythm and sound experience, making it perfect for music enthusiasts and aspiring drummers alike.

Simon Game

This dynamic Simon Game brings the classic memory challenge to life with HTML, CSS, and JavaScript. Through sleek animations and responsive design, users can test their memory skills by repeating random sequences of colors and sounds. It's a fun and engaging project showcasing seamless functionality and vibrant visuals.

Blog API

This Blog API project is a full-stack application that integrates a RESTful API built with Express.js, along with frontend templates rendered using EJS. The backend API handles blog posts with basic CRUD operations (Create, Read, Update, Delete) using Axios to communicate between the frontend and backend.

The project features a user-friendly interface where users can view all posts, create new posts, edit existing posts, and delete posts. Data is managed using a simple in-memory data store, and forms on the frontend allow for seamless interaction. This project demonstrates practical API development, asynchronous operations with Axios, and dynamic content rendering using EJS templates.
